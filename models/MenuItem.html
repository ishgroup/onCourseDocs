MenuItem

  • id integer (optional default: null)
    Unique identifier of menu item
  • title string (optional default: null)
    Title of menu item displayed in the website
  • url string (optional default: null)
    URI for menu link
  • children array (optional default: null)
  • error string (optional default: null)
    Describes reason why menu item cannot be saved.
  • warning string (optional default: null)
    Alert for user. Does not prevent menu from being saved.
