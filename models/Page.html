Page

  • id integer (optional default: null)
    unique id of the page
  • themeId integer (optional default: null)
    theme id applied to the page
  • title string (optional default: null)
    title of the page
  • content string (optional default: null)
    Rich text content
  • visible boolean (optional default: null)
    Is the page visible
  • suppressOnSitemap boolean (optional default: null)
    Is the page suppressed on sitemap
  • urls array (optional default: null)
