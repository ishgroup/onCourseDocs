Theme

A theme is an organisation of blocks which is then applied to a page

  • id integer (optional default: null)
    Unique identifier for this theme. This is not seen in the editor UI.
  • title string (optional default: null)
    Title of theme displayed in the user interface
  • layoutId integer (optional default: null)
    Layout id which this theme uses
  • blocks ThemeBlocks (optional default: null)
  • paths array (optional default: null)
    A list of path patterms to aply the theme
