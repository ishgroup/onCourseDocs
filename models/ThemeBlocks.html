ThemeBlocks

The arrangement of blocks within a theme

  • top array (optional default: null)
    A list of blocks at the top of the page, often used as a page header
  • left array (optional default: null)
    A list of blocks at the left of the page, often used for a sidebar or navigation
  • centre array (optional default: null)
    A list of blocks at the top of the page. They appear above the content.
  • right array (optional default: null)
    A list of blocks at the right of the page
  • footer array (optional default: null)
    A list of blocks at the footer of the page
