Version

A version of the website. One version is considered published at any time and one version is draft at any time.

  • id integer (optional default: null)
    unique id of this version
  • author string (optional default: null)
    First and last name of the user who published this version, or null if still in draft
  • changes integer (optional default: null)
    Count of changes introduced in this version
  • publishedOn DateTime (optional default: null)
    date and time when version was published or null if it is still in draft
  • status VersionStatus (optional default: null)
