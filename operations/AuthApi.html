auth

createSession

Authenticate user by email and password. Returns session token. For all subsequent requests to the API you should ensure a cookie called ESESSIONID is set with this token value.

https://{siteKey}.oncourse.cc/editor/{versionPath}//session

Request body

body LoginRequest (required)
Parameters for request user authorization.

Responses

200 Valid session token String

406 Login unsuccessful. Invalid login name or password.

409 There is more than one user with the same login details.

0 Unexpected error UnexpectedError

Returns

String

Example data

""

destroySession

De-authorise user from editor.

https://{siteKey}.oncourse.cc/editor/{versionPath}//session

Responses

200 Successful logout. The session token is no longer valid.

0 Unexpected error UnexpectedError

Returns

Nothing

getUser

Get current logged in user.

https://{siteKey}.oncourse.cc/editor/{versionPath}//user

Responses

200 Current user User

0 Unexpected error UnexpectedError

Returns

User

Example data

{
