getMenus

Get all menu items.

https://{siteKey}.oncourse.cc/editor/{versionPath}//menu

Responses

200 A list of menu items.

0 Unexpected error UnexpectedError

List

updateMenus

Save menu items.

https://{siteKey}.oncourse.cc/editor/{versionPath}//menu

Request body

body MenuItem (required)
List of menu items which will replace all the menus in the site.

Responses

200 Menus saved. Returns list of menus, some of which may have warnings.

400 Menus not saved. Returns list of menus, some of which may have errors and/or warnings.

0 Unexpected error UnexpectedError

Returns

List
