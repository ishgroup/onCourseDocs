page

createPage

Add new page

https://{siteKey}.oncourse.cc/editor/{versionPath}//page

Responses

200 New page with unique id and URL Page

0 Unexpected error UnexpectedError

Returns

Page

Example data

{
  "urls" : [ {
    "isDefault" : true,
    "link" : "/contact"
  }, {
    "isDefault" : true,
    "link" : "/contact"
  } ],
  "visible" : true,
  "themeId" : 6,
  "id" : 0,
  "title" : "Contact us",
  "suppressOnSitemap" : true,
  "content" : "Phone us on (02) 9550 5001 to find out more."
}

deletePage

Remove page by id.

https://{siteKey}.oncourse.cc/editor/{versionPath}//page/{id}

Path parameters

id (required)
id of page to be deleted

Responses

200 Page successfully deleted

0 Unexpected error UnexpectedError

Returns

Nothing

getPages

Get list of pages.

https://{siteKey}.oncourse.cc/editor/{versionPath}//page

Query parameters

pageLink (optional)
Optional page link to return a list containing just one page

Responses

200 List of pages
