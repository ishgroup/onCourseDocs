redirect

getRedirects

Get redirects for the whole site.

https://{siteKey}.oncourse.cc/editor/{versionPath}//redirect

Responses

200 A list of redirects Redirects

0 Unexpected error UnexpectedError

Returns

Redirects

Example data

{
  "rules" : [ {
    "from" : "/subject/art",
    "to" : "https://github.com",
    "error" : "This URL is already being redirected\n"
  }, {
    "from" : "/subject/art",
    "to" : "https://github.com",
    "error" : "This URL is already being redirected\n"
  } ]
}

updateRedirects

Replace all the redirects for the site with a new list of redirects.

https://{siteKey}.oncourse.cc/editor/{versionPath}//redirect

Request body

body Redirects (required)
List of redirects

Responses

200 List of redirects Redirects

400 Redirects not saved. Returns list of redirects, some of which may have errors and/or warnings. Redirects

0 Unexpected error UnexpectedError

Returns

Redirects

Example data

{
  "rules" : [ {
    "from" : "/subject/art",
    "to" : "https://github.com",
    "error" : "This URL is already being redirected\n"
  }, {
    "from" : "/subject/art",
    "to" : "https://github.com",
    "error" : "This URL is already being redirected\n"
  } ]
}
