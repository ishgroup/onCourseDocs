specialPage

getSpecialPages

Get special page redirects for the whole site.

https://{siteKey}.oncourse.cc/editor/{versionPath}//settings/specialPage

Responses

200 A list of redirects SpecialPages

0 Unexpected error UnexpectedError

Returns

SpecialPages

Example data

{
  "warnings" : [ {
    "message" : "message"
  }, {
    "message" : "message"
  } ],
  "rules" : [ {
    "specialPage" : "Courses",
    "matchType" : "EXACT",
    "from" : "/subject/art",
    "error" : "This URL is already being redirected\n"
  }, {
    "specialPage" : "Courses",
    "matchType" : "EXACT",
    "from" : "/subject/art",
    "error" : "This URL is already being redirected\n"
  } ]
}

updateSpecialPages

Update all special redirects for web site

https://{siteKey}.oncourse.cc/editor/{versionPath}//settings/specialPage

Request body

body SpecialPages (required)
List of special pages

Responses

200 List of special pages SpecialPages

400 Special redirects not saved. Returns list of redirects, some of which may have errors and/or warnings.

0 Unexpected error UnexpectedError

Returns

SpecialPages

Example data

{
  "warnings" : [ {
    "message" : "message"
  }, {
    "message" : "message"
  } ],
  "rules" : [ {
    "specialPage" : "Courses",
    "matchType" : "EXACT",
    "from" : "/subject/art",
    "error" : "This URL is already being redirected\n"
  }, {
    "specialPage" : "Courses",
    "matchType" : "EXACT",
    "from" : "/subject/art",
    "error" : "This URL is already being redirected\n"
  } ]
}
